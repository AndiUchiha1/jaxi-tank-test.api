## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
   yarn install
```

Start the development server

```bash
  yarn start:dev
```
Run build for production

```bash
  yarn build
```

Run production server

```bash
  yarn start
```

  
## Deployment

To deploy this project on vercel run

```bash
  yarn deploy
```
